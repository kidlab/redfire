0.0.0.04 -- May 17th 2011

  • Changed: Native integration for Window OS users only. The audio/video and screenshare now appear in native Spark windows instead of the browser.
  • Changed: New audio/video applications redfire_video.html and redfire_2way.html developed in Flex to replace the old Openlaszlo apps. A big thank you to Mike Grimwade who started work on this.
  • Changed: Support for Flash Player echo cancellation. You do not need a headset anymore.
  • Changed: Support for RTMFP (peer-to-peer). Audio/Video is sent between clients directly without using the Red5 server provided there is a route between clients otherwise an RTMP connection to Redfire is used.
